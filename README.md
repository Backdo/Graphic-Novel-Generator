사용방법


1. 제미니 AI의 유료 API가 필요합니다. (스토리보드, 이미지 생성용을 위해)

2. 타이틀을 프로젝트 제목에 입력하고 소설 텍스트 입력란에 임의의 짧은 이야기나 챗 로그를 작성합니다.

3. 스토리보드 생성 버튼을 누르면 위에 선택된 언어모델을 통해 원본텍스트를 그래픽 노블용 콘티로 변환합니다

4. 각 페이지당 콘티를 필요에 따라 편집하고 내용을 추가할 수 있습니다

5. 콘티 작업이 끝나고 일괄생성버튼을 누르면 각 페이지에 해당하는 그래픽노블을 생성합니다. 다만 유료 API를 이용하니 해당 제미니 계정 요금에 주의하시기 바랍니다.

6. 혹은 각 페이지별로 만들어진 스토리보드를 클립보드에 복사하여 Gemini AI 공식 채팅홈에서 이미지 탭을 눌러 생성가능합니다. 이 경우 제미니 AI의 무료사용량 내에서 이미지 생성을 이용할 수 있습니다.
